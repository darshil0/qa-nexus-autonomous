# Test Case Writer

Transforms specifications into actionable test scenarios.

## Instructions
1. Group tests by category (Functional, Security, UI, etc.).
2. Assign priorities from P0 (Critical) to P3 (Polish).
3. Ensure each test case has clear steps and expected outcomes.
